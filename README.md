🌭 Sabor na Chapa - Landing Page

  Status do Projeto: Em andamento ✅

📋 Sobre o Projeto

Este projeto é uma Landing Page desenvolvida para a lanchonete Sabor na Chapa, localizada em Penha - SC. O objetivo foi criar uma presença digital para o negócio, destacando o cardápio de cachorros-quentes prensados e facilitando o contato direto dos clientes via WhatsApp.

O site foi construído focando em uma navegação fluida, design responsivo e acessibilidade.
🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias fundamentais da web:

- HTML5: Estrutura semântica do conteúdo.
- CSS3: Estilização, layout e responsividade.
- CSS Grid & Flexbox: Para organização dos elementos.
- Variáveis CSS: Para padronização de cores e espaçamentos.
- Media Queries: Para garantir o funcionamento em dispositivos móveis.
- Integração: Botão de chamada para ação (CTA) direto para a API do WhatsApp.

✨ Funcionalidades

- **Cardápio Interativo:** Seção de "Prensados" com descrições detalhadas e preços.
- **Galeria de Fotos:** Exibição visual dos produtos.
- **Botão de Pedido Direto:** Atalho para realizar pedidos diretamente pelo WhatsApp.
- **Responsividade Total:** Interface adaptada para computadores, tablets e smartphones.
